# DailyWallpaper
Android App which sets your wallpaper on a selected interval.

Everything is customizable
- Which photos to select from (Albums, Subsets, All photos)
- How often to set new wallpapers (Each X unlocks, every X hours, etc)
- LockScreen/HomeScreen/Appdrawer etc
 
 I made this app since Google Photos currently has no way of randomly setting wallpapers from its galleries,
 and the background "Widget" has a predetermined set of pictures it cycles through.

Shows Location, Date & Album Name from where the photo was taken, as part of the wallapaper.
